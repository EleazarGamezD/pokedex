## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Ejecutar en desarrollo
```
1.- Clonar el Repositorio
```

2.- Ejecutar 
```
   npm install
```

3.- Tener nest CLI instalado
```
  nom i-g @nestjs/cli
```

4.- Levantar la base de datos 
```  
  docker-compose up -d 
```
